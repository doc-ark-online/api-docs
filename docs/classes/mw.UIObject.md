# UIObject <Badge type="tip" text="Class" /> <Score text="UIObject" />

UI 对象

----------------------------

当你 UIPrefab 制作完成挂载到对象管理器中时，他便成为场景中的一份子，可以使用 GameObject 来查找。

此类中的方法均在客户端调用。
